## Features

* Envelope style budgeting		
* Cross-platform		
* No subscription fees		
* Easy to start and continue using		
* Calculator		
* Private and secure		
* Choice of currency		
* Cloud Sync	 	
* Reporting	 	
* Edit in DataGrid
